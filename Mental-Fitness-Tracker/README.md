# Mental Fitness Tracker

A Python-based Mental Fitness Tracker developed using Object-Oriented Programming principles to monitor mood, stress levels, and sleep patterns.

## Features
- Structured data handling using classes
- Wellness score calculation algorithm
- Categorization of mental health status
- Analytical reporting with insights
- JSON-based persistent storage

## Technologies
- Python
- OOP Concepts
- JSON File Handling
- Algorithmic Analysis

## Run the Application
python main.py
